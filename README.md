# IoT-Based Smart Parking System

A smart parking system built using ESP32-C6, IR sensors, Firebase Realtime Database, LCD display, and a web dashboard.

## Overview

This project detects parking slot availability in real time, controls the entry gate automatically using a servo motor, and updates the parking status on both an LCD screen and an online dashboard.

## Features

- Real-time parking slot detection
- Automatic gate control
- LCD parking status display
- LED slot indicators
- Firebase real-time synchronization
- User dashboard for live slot monitoring
- Admin dashboard for gate control and maintenance mode
- Daily parking usage reports

## Technologies Used

- ESP32-C6
- Arduino / C++
- IR Sensors
- Servo Motor
- LCD I2C
- Firebase Realtime Database
- HTML, CSS, JavaScript
